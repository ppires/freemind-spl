Preferences

Only a small subset of preferences is editable using the graphical interface; you can change them in the Edit / Preferences. At this time, the following preferences can be changed:

  • You can set full antialiasing (smoothing) or antialiasing of edges only. Antialiasing of edges is the default option.
  • You can set a kind of HTML export.

Other preferences can be changed by more technical users in user.properties, which you have to copy into the folder freemind in your home folder.
