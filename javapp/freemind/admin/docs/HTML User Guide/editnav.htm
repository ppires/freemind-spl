Navigating through your mindmaps

Scrolling

You can scroll the map:

  • By using the standard scroll bars.
  • By clicking on the background and dragging it.
  • By using mouse wheel. (To scroll horizontally, hold shift or one of the mouse buttons.)

Zooming

Change the zoom from the toolbar, or zoom in / out from the Edit menu.

As of version 0.7 you can change the zoom by holding down the control key and moving the mouse wheel.

Reordering nodes

Move nodes up and down within a group of siblings (same level) by using Ctrl+Up Arrow or Ctrl+Down Arrow.

Following links

You can follow a link by simply clicking the node.

On a node with a link and children:

  • Clicking at the left side opens the link
  • Clinking at the right side folds or unfolds the tree.

Fast switching among maps

There are several ways to switch from one opened map to another:

  • Right-click on an empty area of the background and pick the map name.
  • Use the toolbar arrows for previous map and next map.
  • Use the Maps menu.
  • Use the Edit menu, and choose Previous Map or Next Map.
  • Use Ctrl+Left Arrow or Ctrl+Right Arrow.
